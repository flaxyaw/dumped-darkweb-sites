PGP Public Keys

There are two types of PGP Public Keys on Archetyp: Auth PGP Public Key and Message PGP Public Key

Auth PGP Public Key: This key is used to identify the vendor, he also uses this Key to login to the market and for 2-FA, you may use this key to encrypt messages and shipping details too.

Message PGP Public Key: This key is only used to encrypt messages and shipping details for the vendor. This key is only setup if the vendor has staff workers, that help him to manage his presence on Archetyp.
